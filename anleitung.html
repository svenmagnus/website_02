User Guide

Private 1:1 video on Tangent Club with optional Lovense toy control (local test and partner remote).

Before each session (short)

  1. Lovense Connect (PC): start the app, power on your toys, confirm they show as connected.
  2. Chrome: Lovense Cam Extension On, cam site test:Tangent-Club selected and status Live (see Lovense setup).
  3. Open tangent-club.com, sign in, open the Lovense floating menu on that tab once (green widget).
  4. Tab Setup → add or pick your partner under Members → Session with → New session → Start instant session.
  5. Start Camera (you go live) or Join Session (orange glow when your partner is already live).
  6. Remote Control (Toys) works once video is connected; use Local Test anytime on your own machine to try toys.

In Setup, status should show Extension ready. If not, use Retry extension connection and open the pink Lovense icon in Chrome or Settings from the floating menu.

Lovense setup (Chrome)

You need Google Chrome, the Lovense Cam Extension, and Lovense Connect on the same computer that runs the stream. For account and cam-site management Lovense recommends Stream Master (lovense.com); the extension is what links toys to the tangent-club.com tab.

1. Connect toys

  1. Install and open Lovense Connect (PC/Mac).
  2. Pair and turn on each toy; they must appear as connected in Connect before you use the website.

2. Cam site test:Tangent-Club (one-time)

While Lovense shows Cam Extension Status: Pending in the developer dashboard, the site name must include the test: prefix. Spelling matters: test:Tangent-Club (colon, capital T and C, hyphen — not test:tangentclub). After Lovense sets status to Active, use Tangent-Club without the prefix.

Stream Master (recommended) — optional but useful for pairing and overview:

  1. Sign in at lovense.com → Stream Master.
  2. Add cam site test:Tangent-Club if it is not listed yet.

Lovense Cam Extension (required for Tangent Club):

  1. Install the Lovense Cam Extension in Chrome.
  2. Open tangent-club.com and sign in, or click the pink Lovense icon in the Chrome toolbar.
  3. Alternatively: on tangent-club.com open the green Lovense floating action menu → Settings.
  4. In the left column under Cam Site, click + and add test:Tangent-Club.
  5. Click that site in the list. The extension must show Live (not offline). Toggle the extension On and confirm Lovense Connect (pc) logged in.

3. Developer dashboard (one-time)

In Lovense Cam Developer, set Model Broadcasting URL Pattern to your site, e.g. *://tangent-club.com/*. When integration is complete and status is still Pending, contact Lovense via the dashboard to start testing.

4. Control on Tangent Club — not Stream Master Basic Levels

Basic Levels in Stream Master (token ranges, reaction time per tip) are for tip-based cam sites. They do not drive the sliders on tangent-club.com.

On Tangent Club, Local Test and Remote Control use direct motor: each connected toy gets its own card; Low / Medium / High / Max / Ultra map to fixed motor strengths; vibration runs until you move the slider to 0 (or tap the active preset again). No duplicate token rows per toy are required.

Special commands (Earthquake, Wave, Pulse, …) still use Lovense’s tip/pattern path with fixed duration — separate from the direct-motor presets. Send Test Tip in the extension is for testing Stream Master rules, not for Tangent direct motor.

If you do use Stream Master Basic Levels elsewhere: ranges are global (you cannot assign the same token band twice, e.g. 1–9 on two rows for two toys). That restriction does not apply to direct motor on Tangent Club.

5. Stream Master rules (one-time)

In Stream Master, open cam site test:Tangent-Club → Rules.

  1. Set Basic Levels to OFF — Tangent Club uses direct motor for sliders and presets (Low / Medium / High / Max / Ultra), not Stream Master token ranges.
  2. Under Special Commands, enable each command below with the exact token value (each command needs its own unique number):
    Command Tokens Status
    Earthquake 100 ON
    Fireworks 120 ON
    Wave 160 ON
    Pulse 200 ON
  3. Click Save (pink button, top right).

What the token values mean: they are not real tips and do not control Low / Medium / High / sliders (direct motor). They are only IDs so Lovense triggers the correct pattern command — internally the site sends a “tip” with exactly that amount.

A pattern runs while its checkbox is checked on tangent-club.com; the page refreshes about every 17 seconds until you uncheck it or move the slider to 0.

Send Test Tip in the Chrome extension tests Stream Master rules — not Tangent Club sliders or presets.

Multiple toys

Every toy connected in Lovense Connect appears as its own card under Local Test, each with its own slider and presets. Your partner sees your toys under Remote Control after you are in a video session (toy list is shared over the session data channel).

List toys in Profile → Lovense toys (owned) so your partner knows what you use (informational only; it does not connect hardware).

1. Account & profile

  1. Register from an invitation: email, 4-digit code, username, password.
  2. Open the verification link in email if prompted.
  3. On Profile: display name, gender, nationality, languages, location, about text, techniques, Lovense toys.
  4. Click Save profile before leaving the tab.

2. Members, invitations & sessions

Tangent Club is built for private 1:1 video between people who know each other — not a public room with many viewers.

  • Everyone can invite — use the account menu (Invite someone or Invite guest by email) to link with another member. Add existing members under Setup → Members.
  • One session = two members. At any moment, a live session has exactly you and one partner (video + optional toy control). There is no third person in the same session.
  • Parallel sessions are fine. Many pairs can be live at the same time on the platform — for example five or more separate 1:1 sessions — each pair uses its own Session ID and does not share a room with others.
  • Either member can Start Camera first; the other Join Session when ready (the join button glows orange when the partner is already live).
  • Model and Admin are optional site flags (managed by the site admin), separate from who invites whom.

3. Your first session

  1. Sign in with username and password.
  2. Messages (header): chat with your partner; history clears when the video session ends.
  3. Under Setup → Sessions & calendar: pick a partner, New session:
    • Start instant session — one side Start Camera; Session ID in Messages.
    • Schedule session for later — date/time; partner notified in chat.
    • Plan in Google Calendar — connect Google Calendar first; synced invite.
  4. Messages panel: drag the title bar; resize from the corner.
  5. Partner: allow camera/mic, then Join Session (Session ID from Messages or Setup).

Broadcast: webcam and OBS (multistreaming)

For a normal private session on Tangent Club, use Webcam & microphone in Setup, pick your devices, then Start Camera. No OBS required.

OBS Studio is optional and useful when you want to multicam / stream on another platform at the same time — for example a polished scene (lighting, layouts, multiple sources) that you push to Tangent Club while also broadcasting elsewhere.

  1. In Setup, under broadcast, choose OBS Studio (WHIP) (not webcam mode).
  2. Copy OBS Server (WHIP) and Bearer Token from the host panel into OBS: Settings → Stream → Service: WHIP, then Start Streaming in OBS.
  3. On the site, Start Camera as host — the browser pulls your OBS feed via WHIP/WHEP and forwards it to your partner over the session (same as webcam, different video source).

Lovense toy control and technique requests stay on tangent-club.com (Remote Control) — not via OBS token overlays from Stream Master, which are built for public tip-based cam sites.

4. Technique requests & toys

  • Technique Requests — only techniques your partner enabled in profile.
  • Remote Control (Toys) — control partner toys when video + data channel are up.
  • Local Test (your toys) — test on your machine; slider commits motor on release; presets apply immediately.

5. Troubleshooting

  • No camera — browser permissions, reload, pick device in Setup.
  • Cannot join — complete New session → instant; partner uses Join Session when you are live.
  • Lovense — Connect running; toys connected; extension On; test:Tangent-Club Live; floating menu opened on tangent-club.com; Extension ready in Setup.
  • Toys do not vibrate on sliders — use Local Test, not only Send Test Tip; check motor hint under the toy card.
  • Server unreachable (dev) — host runs API tunnel (see project docs).
