Gebrauchsanweisung

Kurzanleitung für Gäste und Hosts von Tangent Club (privater Video-Chat mit Lovense-Fernsteuerung).

1. Konto & Profil

  1. Mit Einladung unter Registrieren anlegen (E-Mail, 4-stelliger Code, Benutzername, Passwort).
  2. Falls nötig: E-Mail-Link zur Bestätigung öffnen.
  3. Auf der Willkommensseite Anzeigename, „Über dich“, Techniken und Lovense-Toys eintragen.
  4. Profil speichern, dann Zum Video-Stream.

2. Raum betreten

  1. Auf der Startseite mit Benutzername + Passwort anmelden (Raum-Zugang).
  2. Kamera/Mikrofon im Browser erlauben.
  3. Als Gast: Host-Peer-ID einfügen und verbinden. Als Host: „Start as Host“.

3. Lovense (Chrome)

  1. Lovense Cam Extension installieren (Google Chrome).
  2. In der Extension die Site test:Tangent-Club hinzufügen.
  3. Toys in der Extension verbinden — auf der Stream-Seite muss das Lovense-Widget erscheinen.
  4. Status „Extension bereit“ abwarten. Toys im Profil textlich notieren (Info für den Partner).
  5. Broadcast-URL im Lovense Developer Dashboard als „Model Broadcasting URL Pattern“ eintragen.

4. Technik-Anfragen & Toys

  • Rechts Tab Technique Requests: nur Techniken, die der Partner im Profil freigegeben hat.
  • Tab Remote Control (Toys): Intensität / Impulse an verbundene Lovense-Geräte des Partners.

5. Fehlerbehebung

  • Keine Kamera: Browserberechtigung prüfen, Seite neu laden.
  • Verbindung scheitert: Peer-ID prüfen, beide online, VPN/Firewall testweise aus.
  • „Server nicht erreichbar“: Host muss API-Tunnel laufen haben (Cloudflare → Port 8787).
