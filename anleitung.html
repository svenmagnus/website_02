User Guide

Quick start for guests and hosts on Tangent Club (private video chat with Lovense remote control).

Reihenfolge vor dem Stream (Kurz)

In Tangent Club gibt es keinen Lovense-Wizard. Die Einrichtung passiert in Stream Master und der Chrome-Erweiterung. Danach diese Reihenfolge:

  1. Lovense vorbereiten (vor der Session): Lovense Connect (PC) — Toy verbinden (z. B. Diamo). Chrome-Erweiterung ein, Cam-Site test:Tangent-Club in Stream Master gespeichert (Regeln optional). Developer-Dashboard: URL-Muster = Ihre Tangent-Club-Adresse (z. B. https://tangent-club.com/).
  2. Anmelden auf tangent-club.com.
  3. Tab Setup → unter Members Partner hinzufügen oder wählen → Session with Partner auswählen.
  4. New session → Start instant session.
  5. Wer zuerst live geht: Start Camera (Kamera/Mikro erlauben). Partner: Join Session (leuchtet orange, wenn der andere bereits live ist).
  6. Remote-Toys: Tab Remote Control (Toys) — erst sinnvoll, wenn Video verbunden ist.

Wenn die Erweiterung „nicht aufgeht“: Popup über das Lovense-Icon in Chrome öffnen (nicht Stream Master). Im Setup-Tab Retry extension connection und Status „Extension ready“ prüfen.

Mehrere Toys (z. B. 4 Geräte)

Tangent Club (Local Test / Remote): Jedes angeschlossene Toy bekommt eine eigene Karte und einen eigenen Regler — Direct Motor, bis Regler 0. Keine doppelten Token-Bereiche nötig.

Stream Master Basic Levels: Token-Stufen sind global (nicht pro Toy). Du kannst nicht für Lush und Diamo jeweils „1–9 Tokens“ eintragen — Lovense meldet: You cannot use the same number for more than one level. Entweder:

  • Eine Tip-Tabelle mit einem Toy (oder Lush,Diamo in einer Zeile für Test Tip), oder
  • Basic Levels für Tangent ignorieren und nur Direct Motor auf tangent-club.com nutzen (empfohlen für „bis Regler 0“).

1. Account & Profile

  1. Create your account from an invitation via Register (email, 4-digit code, username, password).
  2. If needed, open the email verification link.
  3. On the profile screen, set display name, gender, nationality, languages, location, about text, techniques, and Lovense toys.
  4. Click Save profile at the bottom of the profile tab before you leave.

1b. Host, Guest & Invitations

  • Guest accounts are usually passive in a session (they connect to a host’s stream). In your profile and in the admin view, status is shown as Guest or Host.
  • Guests may still invite other people by email (account menu → Invite guest by email). Sending an invite can turn or keep your account as a Host for invitations.
  • Host accounts can start a stream and invite guests.
  • Model and Admin are separate flags (site admin manages them).

2. Your First Session (like Google Meet)

  1. Sign in with username + password.
  2. Messages (header icon): chat with your host or guest — history is cleared when the video session ends (Hang up or disconnect).
  3. Everyone is a model — invite others into your personal model pool. Under Setup → Sessions & calendar, choose a model, then New session:
    • Start instant session — one partner clicks Start Camera; Session ID is sent in Messages.
    • Schedule session for later — pick date/time; partner is notified in chat.
    • Plan in Google Calendar — connect Google Calendar first; creates a synced event with email invites for both sides.
  4. Messages (header): floating window — drag the title bar to move, resize from the corner.
  5. Partner: Session ID appears in Messages or Setup (auto-filled when possible), allow camera/mic, then Join Session.

3. Lovense (Chrome) — no in-app wizard

Stream Master (lovense.com) is for rules and toy pairing. The Cam Extension runs on the Tangent Club tab during the stream. Tangent Club only shows status and remote control — it does not open Lovense’s setup wizard.

  1. Install Lovense Cam Extension in Google Chrome (not Safari/Firefox for the extension).
  2. In Stream Master: add cam site test:Tangent-Club (exact spelling, colon). Connect toy via PC or mobile; save rules if you use tip levels (optional for private 1:1).
  3. On stream day: extension popup On, “Lovense Connect (pc) Logged in”, toy listed. Open tangent-club.com → Setup → status Extension ready.
  4. List toys in your Profile so your partner sees them under Remote Control.
  5. Lovense Developer: Model Broadcasting URL Pattern = your site origin + path (e.g. https://tangent-club.com/*).

4. Technique Requests & Toys

  • Right tab Technique Requests: only techniques your partner enabled in profile.
  • Tab Remote Control (Toys): intensity and pulses for connected partner toys.

5. Troubleshooting

  • No camera: check browser permissions, then reload.
  • Connection fails: complete New session → instant first; partner uses Join Session when you are live.
  • Lovense: extension on same browser profile; retry in Setup; toy connected in Connect before opening the site.
  • “Server unreachable”: host must run API tunnel (Cloudflare → port 8787).
