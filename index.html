Lovense — before you stream

There is no wizard inside Tangent Club. Configure toys once in Stream Master and the Chrome extension, then follow this order every session.

  1. One-time: Chrome + Lovense Cam Extension. In Stream Master add site test:Tangent-Club (same spelling). In Lovense Developer register URL pattern
  2. Stream Master: In Basic Levels choose one toy per row (e.g. one row Diamo, one row Lush) — not Lush,Diamo in one dropdown. Needed for Test Tip / tokens only.
  3. Every session: Lovense Connect (PC) — toys on. Extension On, test:Tangent-Club. Open the widget on tangent-club.com once.
  4. Open this Setup tab — status below must show Extension ready (click the Lovense widget on the page if needed).
  5. Members — add or pick your partner. Session with → New session → Start instant session.
  6. Start Camera (you go live) or Join Session (partner already live — button glows orange).

What is Tangent Club?

A private 1:1 video lounge — not a public cam site. You connect only with people you invite (or who invited you). One partner starts the camera; the other joins the session with the shared Session ID. Optional Lovense toy control and technique requests make it interactive — but there is no public gallery, tipping room, or anonymous browsing like on typical adult platforms.

Start as host: (1) Sign in on the start screen → (2) Choose broadcast Webcam & microphone for normal streaming without OBS → (3) Pick camera/mic → Start Camera. For OBS scenes: broadcast OBS (WHIP). Invite models from the account menu. Either side can Start Camera or Join Session.
