Account & login

  • Invitation only — register via the link or 4-digit code from your partner. Open register with a valid token.
  • Email verification — if required, use the link in the registration email before logging in.
  • Forgot password — ask the person who invited you or a site administrator to reset your account.
  • SMTP / invites — hosts can configure outgoing mail under account menu → email server (admin). Without SMTP, share the invite link manually from the invite dialog.

Sessions & video

  • One session = two people. Add your partner under Setup → Members, choose Session with, then New session → Start instant session.
  • Either side can Start Camera; the other uses Join Session (orange when the partner is already live).
  • No picture / black video — allow camera and microphone in the browser, reload, pick the correct device in Setup. For OBS/WHIP, refresh the bearer token and restart the stream.
  • Cannot connect — both must complete New session (instant) first; check Messages for the Session ID; try another browser profile without heavy extensions.
  • Partner busy — your partner may already be in another 1:1 session; wait and try again.

Lovense toys

  • Use Google Chrome, Lovense Connect (PC), and the Lovense Cam Extension on the same machine.
  • Cam site test:Tangent-Club must be Live in the extension (Stream Master + extension popup).
  • On tangent-club.com open the Lovense floating menu once; Setup should show Extension ready.
  • Local Test — one card per toy; slider commits on release; use preset buttons for quick tests. Send Test Tip in the extension is not the same as direct motor on the site.
  • Remote Control — available when video and the data channel are connected; toys are shared between partners automatically.

Stream Master Basic Levels do not control Tangent sliders — see the user guide.

Appearance & browser

  • Theme: account menu or Settings → Light, Dark, or Hippie. Saved in localStorage under key theme.
  • Hard refresh after updates: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows).
  • Chat history clears when the video session ends (hang up or disconnect).
