Benutzerhandbuch – One-to-One-Videochat-App
1. Zweck der Anwendung
Die Anwendung ermöglicht einen direkten privaten Videochat zwischen zwei Personen über eine Peer-to-Peer-Verbindung.
Es gibt:
    • einen Host
    • einen Gast / Teilnehmer
Der Host erstellt die Verbindung. 
Der Teilnehmer verbindet sich über die Peer-ID des Hosts.

2. Voraussetzungen
Empfohlene Browser:
    • Google Chrome
    • Microsoft Edge
Erforderlich:
    • Kamera
    • Mikrofon
    • stabile Internetverbindung

3. Oberfläche der App
Die Oberfläche besteht aus mehreren Bereichen:
Lokales Kamerabild („DU“)
Zeigt das eigene Kamerabild.
Gegenüber („REMOTE“)
Zeigt den verbundenen Gesprächspartner.
Verbindungsbereich
Hier wird die Verbindung aufgebaut:
    • Als Host starten
    • Mit Host verbinden
    • Trennen
Peer-ID
Jeder Host erhält automatisch eine eindeutige Peer-ID.
Diese muss an den Gesprächspartner weitergegeben werden.

4. Verbindung als Host herstellen
Schritt-für-Schritt
    1. Öffne die App:
Videochat App
    2. Kamera- und Mikrofonzugriff erlauben.
    3. Klicke auf:
„Als Host starten“
    4. Unten erscheint:
„Deine aktuelle Peer-ID“
    5. Sende diese Peer-ID an den Teilnehmer.
    6. Warte auf die eingehende Verbindung.

5. Verbindung als Teilnehmer herstellen
    1. Öffne den Einladungslink.
    2. Kamera- und Mikrofonzugriff erlauben.
    3. Klicke:
„Mit Host verbinden“
    4. Füge die Peer-ID des Hosts in das Eingabefeld ein.
    5. Die Verbindung wird automatisch aufgebaut.

6. Ansichtsmodi
Die App unterstützt verschiedene Darstellungen:
    • Nebeneinander
    • PiP Gegenüber groß
    • PiP Du groß
Zusätzlich kann die Position des PiP-Fensters gewählt werden:
    • unten rechts
    • andere Positionen je nach Einstellung

7. Lovense Cam Extension (Chrome)
Integration über die Lovense Cam Extension (nicht die Lovense-Connect-App allein):
    1. Google Chrome verwenden und die Lovense Cam Extension installieren.
    2. In der Extension die Cam-Site Tangent-Club hinzufügen.
    3. Toys in der Extension verbinden — auf der Broadcast-Seite muss das Lovense-Widget erscheinen.
    4. Status „Extension bereit“ abwarten, dann „Test: receiveTip(25)“ oder Fernsteuerung nutzen.
    5. Die Broadcast-URL muss im Lovense Developer Dashboard als „Model Broadcasting URL Pattern“ eingetragen sein.


Fernsteuerung / Impulsfunktion
Im rechten Bereich kann der Teilnehmer Signale an den Host senden.
Möglichkeiten:
    • Intensität einstellen
    • verschiedene Stufen:
        ◦ Leicht
        ◦ Mittel
        ◦ Stark
        ◦ Max
    • Impuls senden

8. Verbindung trennen
Zum Beenden:
    • auf „Trennen“ klicken
    • Browserfenster schließen

9. Fehlerbehebung
Keine Kamera sichtbar
    • Browserberechtigung prüfen
    • Seite neu laden
Kein Ton
    • Mikrofon aktivieren
    • Audioausgabe prüfen
Verbindung klappt nicht
    • richtige Peer-ID verwenden
    • beide Teilnehmer online
    • Firewall/VPN testweise deaktivieren

10. Sicherheitshinweis
Die Anwendung ist ausschließlich für private und einvernehmliche Nutzung vorgesehen.

User Manual – One-to-One Video Chat App
1. Purpose of the Application
This application allows a direct private peer-to-peer video chat between two people.
There are two roles:
    • Host
    • Guest / Participant
The host creates the session.
The participant connects using the host’s Peer ID.

2. Requirements
Recommended browsers:
    • Google Chrome
    • Microsoft Edge
Required:
    • Camera
    • Microphone
    • Stable internet connection

3. App Interface
The interface contains several sections:
Local Camera View (“YOU”)
Displays your own camera feed.
Remote View (“REMOTE”)
Displays the connected participant.
Connection Area
This section is used to establish the connection:
    • Start as Host
    • Connect to Host
    • Disconnect
Peer ID
Each host automatically receives a unique Peer ID.
This ID must be shared with the participant.

4. Starting a Session as Host
Step-by-Step
    1. Open the app:
Videochat App
    2. Allow camera and microphone access.
    3. Click:
“Start as Host”
    4. Your current Peer ID will appear at the bottom.
    5. Send this Peer ID to the participant.
    6. Wait for the incoming connection.

5. Joining as Participant
    1. Open the invitation link.
    2. Allow camera and microphone access.
    3. Click:
“Connect to Host”
    4. Paste the host’s Peer ID into the input field.
    5. The connection will be established automatically.

6. View Modes
The app supports different layouts:
    • Side-by-side
    • PiP Remote Large
    • PiP You Large
You can also choose the Picture-in-Picture position:
    • Bottom right
    • Other positions depending on settings

7. Lovense Cam Extension (Chrome)
Use the Lovense Cam Extension (not the Lovense Connect app alone):
    1. Use Google Chrome with the Lovense Cam Extension installed.
    2. Add cam site Tangent-Club in the extension.
    3. Connect toys in the extension — the Lovense widget must appear on the broadcast page.
    4. Wait for “Extension ready”, then use “Test: receiveTip(25)” or remote control.
    5. Register the broadcast URL in the Lovense Developer Dashboard (Model Broadcasting URL Pattern).
    5. Once connected, the remote control / impulse features in the video chat app can be used.
Remote Control / Impulse Function
On the right side, participants can send signals to the host.
Available options:
    • Adjust intensity
    • Different levels:
        ◦ Light
        ◦ Medium
        ◦ Strong
        ◦ Max
    • Send impulse

8. Ending the Session
To end the session:
    • Click “Disconnect”
    • Or close the browser window

9. Troubleshooting
Camera not visible
    • Check browser permissions
    • Reload the page
No audio
    • Enable microphone access
    • Check audio output settings
Connection fails
    • Verify the correct Peer ID
    • Ensure both users are online
    • Temporarily disable VPN or firewall if needed

10. Safety Notice
This application is intended exclusively for private and consensual use.
